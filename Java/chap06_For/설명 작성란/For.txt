for문
	자바에서 반복문 중 하나로 일정 횟수 동안 반복 작업을 수행하는데 사용
	주로 반복 횟수가 정확하게 알려져 있는 경우 사용
	박복문 내에서 카운터 변수를 사용하여 반복을 제어하는 형태로 많이 사용
	
	for문의 구조:
		for(초기화; 조건식; 증감 부분){
			//조건이 참일 때 반복해서 실행될 코드
		}
		초기화: 반복문이 시작될 때 최초 1회만 실행되는 부분
		반복문의 카운터 변수를 초기화
		조건식: 반복문의 실행여부를 결정하는 조건 체크
		조건이 참일 경우 반복문이 실행 
		거짓일 경우 반복문이 종료
		증감부분: 각 반복문이 끝날 때마다 실해외는 부분
		카운터 변수의 값을 증가 또는 감소
